Planifiqator este o aplicatie web pentru planificari de calatorii, facuta in .NET si Angular. Scopul acestei aplicatii este de a oferi romanilor oferte de vacanta, de a fi mai organizat cu planurile de vacanta si de a planifica vacantele. Aplicatia foloseste tehnica de gamification pentru a incuraja utilizarea ei.
